★ 主な機能

支援者データの管理・分析に必要な機能をポップアップに凝縮しました。

🔎

ワンクリック抽出

FANBOX支援者一覧ページから、支援者名・ユーザーID・プラン名・支援開始日・メモを自動抽出します。

📈

テーブル表示 · ソート

抽出結果をポップアップ内にテーブルで表示。各列のヘッダーをクリックして昇順・降順でソートできます。

🔍

リアルタイムフィルター

プラン名（複数選択可）、支援開始日の範囲、継続月数でデータをリアルタイムに絞り込めます。

💾

CSV/JSONエクスポート

フィルター適用後のデータをCSVまたはJSON形式でダウンロード。Excelでそのまま開けるBOM付きCSVに対応。

🛡

対象ページ限定動作

FANBOXの支援者管理ページでのみアイコンが有効化。安全に使えるManifest V3準拠の設計です。

🛠

デバッグ支援

FANBOXの構造変更で動作しない場合、ページのHTMLをダウンロードして開発者に提供できます。

💻 ポップアップ UI

拡張機能アイコンをクリックすると表示されるポップアップを再現しました。実際の外観・機能を確認できます。

シンプルで強力な操作画面

左のUIは実際のChrome拡張機能ポップアップを再現したものです。全ての操作がこの1画面で完結します。

  • ▲
    抽出ボタン — ワンクリックでFANBOX支援者ページからデータを自動スクレイピング。ページのDOM構造を解析して情報を取得します。
  • ■
    フィルター — プラン名のチェックボックスと日付/月数フィルターでリアルタイムに絞り込み。変更は即座にテーブルに反映されます。

    ソート可能テーブル — 列ヘッダーをクリックするだけで昇順/降順ソート。交互背景色で見やすいデザインです。
  • ↧
    エクスポート — CSV（BOM付きでExcel互換）またはJSON形式でダウンロード。タイムスタンプ付きファイル名で自動生成されます。

🔧 インストール方法

Chromeウェブストア未公開のため、手動でインストールしてください。

  1. 1
    パッケージをダウンロード・解凍

    GitHub Releasesから最新のZIPファイルをダウンロードし、任意の場所に解凍します。

  2. 2
    Chrome拡張機能ページを開く

    アドレスバーに入力して拡張機能の管理ページを開きます。

    chrome://extensions
  3. 3
    デベロッパーモードを有効化

    右上のトグルスイッチをオンにします。

    デベロッパーモード
  4. 4
    「パッケージ化されていない拡張機能を読み込む」をクリック

    解凍したフォルダ（manifest.json が含まれるフォルダ）を選択します。

  5. 5
    インストール完了

    「FANBOX支援者リスト抽出ツール」が一覧に表示されたら完了です。FANBOXの支援者管理ページを開くとアイコンが有効化されます。

注意: 手動インストールした拡張機能は、Chrome再起動時に無効化されることがあります。その場合は再度有効化してください。

重要: このツールはPixiv FANBOXのHTML構造に依存しています。FANBOX側のアップデートにより構造が変更された場合、正常に動作しなくなる可能性があります。その場合はツールのアップデートをお待ちください。
