Blog Card Builder

ECサイトの商品ページから情報を自動取得し、はてなブログに貼れるHTMLカードを一発生成するChrome拡張機能

● Amazon ● AliExpress ● Sunstella ● OGP対応サイト

★ 主な機能

商品リンクカードの作成に必要な全てを、ブラウザのサイドパネルに凝縮しました。

🔎

ワンクリックで情報取得

閲覧中の商品ページからタイトル・画像・URLを自動スクレイピング。手入力不要です。

🏠

複数ECサイト対応

Amazon、AliExpress、Sunstellaに専用パーサーを搭載。その他のサイトもOGPフォールバックで対応。

🔗

アフィリエイトリンク自動生成

Amazonアソシエイトタグの付与やSunstellaアフィリエイトURLへの変換を自動で処理します。

🎨

ボタン色のカスタマイズ

各ショップボタンの色をカラーピッカーで自由に設定。ブログのデザインに合わせられます。

📄

HTML/CSSコード生成

はてなブログのテーマに互換のあるHTMLとCSSを自動生成。コピペで即利用可能。

🕒

履歴管理（最大50件）

作成したカードを自動保存。過去のカードをワンクリックで復元・再利用できます。

💻 サイドパネル UI

Chromeのサイドパネルとして動作します。実際のUIを操作して確認してください。

3つのタブで完結する操作

左のUIは実際の拡張機能の画面を再現したものです。下のタブを切り替えて、各画面の雰囲気を確認できます。

  • B
    Builder タブ — 商品ページの情報を取得し、タイトルの編集やショップリンクの追加を行います。完成したらHTMLコードをワンクリックでコピー。
  • H
    History タブ — 作成済みカードを最大50件保存。サムネイル・タイトル・対応ショップのタグが一覧で確認でき、ワンクリックで復元できます。
  • C
    Config タブ — アフィリエイトIDの設定と、ショップボタンの色カスタマイズ。CSSコードの自動生成＆コピーもここで完結します。

Blog Card Builder

※ ビルド済みパッケージは GitHub Releases ページからダウンロードできます

🔧 インストール方法

Chromeウェブストア未公開のため、手動でインストールしてください。

  1. 1
    パッケージをダウンロード・解凍

    GitHub Releasesから最新のZIPファイルをダウンロードし、任意の場所に解凍します。

  2. 2
    Chrome拡張機能ページを開く

    アドレスバーに入力して拡張機能の管理ページを開きます。

    chrome://extensions
  3. 3
    デベロッパーモードを有効化

    右上のトグルスイッチをオンにします。

    デベロッパーモード
  4. 4
    「パッケージ化されていない拡張機能を読み込む」をクリック

    解凍したフォルダ（manifest.json が含まれるフォルダ）を選択します。

  5. 5
    インストール完了

    「Blog Card Builder」が一覧に表示されたら完了です。ツールバーのアイコンをクリックするとサイドパネルが開きます。

注意: 手動インストールした拡張機能は、Chrome再起動時に無効化されることがあります。その場合は再度有効化してください。

⚙ 初回設定

インストール後、最初にConfigタブでアフィリエイト情報を設定してください。

A

Amazon

Amazonアソシエイトのタグ（例: your-tag-22）を入力してください。商品URLに自動で ?tag=... が付与されます。

S

Sunstella

アフィリエイトのベースURLを入力します。商品のパスが自動追加され、アフィリエイトリンクが生成されます。

🎨

CSSの設定

カラーピッカーで各ショップボタンの色を調整し、生成されたCSSコードをコピーして、はてなブログの「デザインCSS」に一度だけ貼り付けてください。
